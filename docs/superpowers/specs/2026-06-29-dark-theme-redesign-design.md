# 暗夜金融 — 全局 UI 重设计

## 目标

将记账工具从"功能可用但像 Demo"的状态，升级为有独立视觉语言的暗色主题产品。核心提升点：**排版节奏 > 质感细节 > 品牌色识別度**。

## 设计方向

**暗夜金融 (Dark Finance)** — 深蓝黑底 + 玻璃质感卡片 + 青蓝强调，融合 FinTech 的效率感与个人记账的生活感。

---

## 1. Design Tokens

### 1.1 色彩

| Token | 色值 | 用途 |
|---|---|---|
| `--bg-root` | `#0a0e14` | 页面根背景 |
| `--bg-surface` | `rgba(30, 41, 59, 0.55)` | 卡片/面板背景 |
| `--bg-surface-hover` | `rgba(30, 41, 59, 0.75)` | 卡片悬停 |
| `--border-subtle` | `rgba(71, 85, 105, 0.25)` | 卡片边框 |
| `--border-default` | `rgba(71, 85, 105, 0.35)` | 输入框/分隔边框 |
| `--text-primary` | `#e2e8f0` | 主文字（slate-200） |
| `--text-secondary` | `#94a3b8` | 次要文字（slate-400） |
| `--text-muted` | `#64748b` | 辅助/占位（slate-500） |
| `--text-disabled` | `#5c6a7a` | 禁用/更深辅助 |
| `--accent` | `#38bdf8` | 主强调色（sky-400） |
| `--accent-strong` | `#0284c7` | 渐变起点/深色强调 |
| `--accent-glow` | `rgba(2, 132, 199, 0.25)` | 按钮发光阴影 |
| `--expense` | `#f87171` | 支出（暖红，red-400） |
| `--income` | `#34d399` | 收入（emerald-400） |
| `--warning` | `#fbbf24` | 预算预警（amber-400） |
| `--danger` | `#ef4444` | 超支危险 |

### 1.2 排版

- **字体族**：系统原生字体栈 `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- **数字**：全局 `font-variant-numeric: tabular-nums`，金额类 weight 650-750
- **字重**：标题 700-750，正文 400-500，辅助 400-500

| 层级 | 字号 | 字重 | 行高 | 用途 |
|---|---|---|---|---|
| H1 | 26px | 700 | 1.15 | 页面标题 |
| H2 | 18px | 650 | 1.25 | 区块标题 |
| H3 | 13px | 600 | 1.3 | 卡片内标题 |
| 数据大 | 22px | 750 | 1.0 | 统计卡片数字 |
| 数据中 | 20px | 700 | 1.0 | 二级数字 |
| 正文 | 14px | 450 | 1.5 | 记录项主文字 |
| 正文小 | 12px | 450 | 1.4 | 记录项次要 |
| 辅助 | 11px | 500 | — | 标签、进度文字 |
| 微小 | 10-10.5px | 500 | — | 卡片标签（uppercase） |
| 导航 | 9px | 500/600 | — | 底部 Tab |

### 1.3 间距

基于 4px 网格：
- **页面内边距**：20px 水平，28px 顶部
- **模块间距**：12px 小模块，18px 大模块
- **卡片内边距**：16-18px
- **Grid 间距**：10px
- **元素内间距**：6-8px（标签与数据），10-14px（区块标题与内容）

### 1.4 圆角

- 大卡片/面板：`14px`
- 按钮：`14px`
- 输入框：`12px`
- 进度条：`99px`（全圆）
- 标签/徽章：`99px`

### 1.5 阴影与质感

- 玻璃卡片：`backdrop-filter: blur(12px)` + 半透明底 + 1px 微弱边框
- 主按钮发光：`box-shadow: 0 4px 20px rgba(2, 132, 199, 0.25)`
- 不使用传统 `box-shadow` 层级阴影，依靠边框和背景透明度区分层级

## 2. 组件系统

### 2.1 卡片 `.card`

```css
.card {
  background: rgba(30, 41, 59, 0.55);
  border-radius: 14px;
  border: 1px solid rgba(71, 85, 105, 0.25);
  backdrop-filter: blur(12px);
  padding: 16px 18px;
}
```

### 2.2 输入框

```css
.input {
  background: rgba(30, 41, 59, 0.4);
  border: 1px solid rgba(71, 85, 105, 0.35);
  border-radius: 12px;
  color: #e2e8f0;
  padding: 12px 16px;
  font-size: 14px;
}
.input:focus {
  border-color: #38bdf8;
  outline: none;
}
```

### 2.3 按钮

- **主按钮**：渐变 `#0284c7 → #0ea5e9` + 发光阴影，白色文字，weight 650
- **次按钮**：透明底 + `rgba(71,85,105,0.35)` 边框 + 青色文字
- **危险按钮**：透明底 + 红色边框/文字
- **禁用**：`bg-disabled + text-disabled`，降低透明度

### 2.4 底部导航栏

- 背景：`rgba(10, 14, 20, 0.85)` + `backdrop-filter: blur(16px)`
- 上边框：`1px solid rgba(71, 85, 105, 0.15)`
- 图标：emoji 保留（用户习惯），尺寸 18px
- 选中态：`--accent` 颜色，未选中：`--text-disabled`
- 标签：9px，weight 500，选中态 weight 600

### 2.5 进度条

- 高度：6px（关键进度如总预算）/ 4px（分类进度）
- 底色：`rgba(71, 85, 105, 0.25)`
- 填充：默认 `--accent` 渐变，超 80% `--warning`，超 100% `--danger`
- 圆角：全圆 `99px`

### 2.6 Tab 切换（支出/收入）

- 容器：`rgba(20, 30, 44, 0.6)` 深色底
- 选中：`rgba(30, 41, 59, 0.8)` + 1px 青色边框
- 未选中：无背景，`--text-muted`
- 过渡：`transition: all 0.2s ease`

### 2.7 Toast

- 保持顶部悬浮，玻璃质感
- 成功：青色左边框 + 半透明蓝底
- 警告：琥珀左边框
- 危险：红色左边框

## 3. 逐页设计

### 3.1 概览 Dashboard
**核心改动**：信息层级重构
- 顶部：标题 + 月份徽章（pill badge），去掉"概览"的大标题？→ 保留，加大
- 三张统计卡：支出/收入/结余，数字是视觉焦点，标签 uppercase 缩小
- 预算进度条：显示已用/总额/剩余三行信息，进度条加粗到 6px
- 本周概览：三列 + 竖线分隔，每列加笔数
- 分类预算：进度条组，带分类色点
- 异常提醒：黄色/红色左边框卡片
- 备份提醒：蓝色左边框卡片
- "记一笔"按钮：渐变 + 发光阴影，页面底部

### 3.2 记账 AddRecord
**核心改动**：字段间距 + 视觉权重
- 支出/收入 Tab 切换：深色底胶囊切换
- 金额输入（AmountInput）：更突出的视觉权重，大号数字
- 分类选择：网格展示，选中态青色边框高亮
- 日期/支付方式/备注：输入框统一样式
- 模板操作：次按钮样式
- 保存按钮：主按钮，固定底部或跟随表单

### 3.3 流水 Records
**核心改动**：列表质感 + 搜索体验
- 标题行：标题 + 搜索图标 + 年/月/类型/支付方式下拉
- 年月选择器：暗色 select，圆角 12px
- 汇总栏：共 N 笔 / 支出 / 收入，一行
- 日期分组：日期标题 + 日合计
- 记录行：玻璃卡片，图标 + 分类名/备注 + 金额/支付方式 + 编辑/删除操作
- 编辑模式：蓝底高亮卡片
- 搜索模式：返回 + 搜索框 + 结果数

### 3.4 日历 Calendar
**核心改动**：月视图视觉升级
- 月份切换：左右箭头 + 年月标题
- 日期格子：暗色底，当日高亮青色圆环
- 有记录日期：圆点标记，支出=红色点，收入=绿色点
- 底部：选中日期的记录清单

### 3.5 报表 Reports
**核心改动**：图表配色适配暗色
- 年月选择器同流水页
- 月度汇总卡
- 图表：Recharts 暗色 theme（坐标轴/网格线/tooltip 均适配暗底）
- 饼图下钻 + 趋势折线图
- 异常预警卡

### 3.6 预算设置 Budget
**核心改动**：表单统一
- 总预算设置：输入框 + 保存按钮
- 分类预算列表：图标 + 分类名 + 输入框 + 保存按钮
- 输入框统一样式

### 3.7 我的 Settings
**核心改动**：列表项质感
- 分组列表，每组带标题
- 列表项：左侧图标 + 标题/描述 + 右侧箭头/操作
- 数据管理：导出/导入/清空按钮组
- 关于：版本号等

## 4. TailwindCSS 配置变更

```js
// tailwind.config.js 扩展颜色
colors: {
  'bg-root': '#0a0e14',
  'bg-surface': 'rgba(30, 41, 59, 0.55)',
  'text-primary': '#e2e8f0',
  'text-secondary': '#94a3b8',
  'text-muted': '#64748b',
  'accent': '#38bdf8',
  'accent-strong': '#0284c7',
  'expense': '#f87171',
  'income': '#34d399',
  // 保留旧 token 名用于渐进迁移：
  // paper → bg-root
  // ink → text-primary
  // border → border-subtle
}
```

**不删除旧 token**，通过替换引用来渐进迁移，旧 token 标记 deprecated。

## 5. 实施策略

### 5.1 改造顺序
1. **Design tokens + CSS 变量** → 全局生效
2. **Layout 底部导航** → 最先可见的框架
3. **Dashboard 概览** → 首页，最高频
4. **AddRecord 记账** → 第二高频
5. **Records 流水** → 第三高频
6. **Reports 报表** → 图表适配
7. **Calendar 日历 + Budget 预算 + Settings 设置**

### 5.2 实施原则
- 不改变现有功能和交互逻辑
- 不改变数据流（pages → hooks → adapters）
- 只改视觉层：Tailwind class + CSS
- 每个页面改完立即用 `/verify` 验证
- 保留 emoji 图标（不做 SVG 图标迁移），保持轻量

---

## 6. 不做什么

- ❌ 不引入新字体文件（用系统字体栈）
- ❌ 不引入新动画库
- ❌ 不替换 Recharts（只改配色）
- ❌ 不改功能逻辑
- ❌ 不做 SVG 图标系统（emoji 够用）
- ❌ 不加毛玻璃效果到每个元素（只在卡片用）
